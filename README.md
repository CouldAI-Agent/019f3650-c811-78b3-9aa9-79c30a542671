# Bhavin Wheels Pvt Ltd - Loan Sanctioning & CSAT Study

This application provides a comprehensive digital platform for analyzing the **Loan Sanctioning Process** and **Customer Satisfaction** at the Bhavin Wheels Pvt Ltd Mahindra showroom. It supports tracking loan applications, statuses, and gathering customer feedback to improve showroom operations.

## Features
- **Dashboard:** A responsive overview of loan application metrics and customer satisfaction scores.
- **Loan Management:** View and track the status of customer vehicle loan applications (Pending, Approved, Rejected).
- **Customer Feedback:** A dedicated module for collecting and analyzing customer satisfaction data regarding the loan process.

## Tech Stack
- Flutter (Cross-platform support for Web, iOS, Android, and Desktop)
- Material 3 Design

## Setup
1. Ensure you have the [Flutter SDK](https://flutter.dev/docs/get-started/install) installed.
2. Clone the repository and run `flutter pub get`.
3. Run the app using `flutter run`.
